# awesome-ai-knowledge
Awesome AI Knowledge - 最全的AI 人工智能知识库

> Curated full-stack technical knowledge base for LLM / Multimodal AIGC / AI Agent
Open Source · MIT License

## 仓库简介
本仓库聚焦当前四大AI核心赛道：**大语言模型LLM、多模态AIGC、AI Agent智能体、Agent 工程与训练知识库**，统一采用标准化8段式全链路结构梳理技术内容，覆盖架构、数据集、数据合成、训练、微调、评测、开源落地全流程，消除碎片化学习。

### 层级规范
1. 大语言模型LLM / AI Agent智能体：一级分类 → 8个标准二级技术模块
2. 多模态AIGC：一级分类 → 6大垂直细分二级赛道 → 统一8套三级技术模块
3. 全仓库通用固定8大内容模块：
- 01 模型基本结构
- 02 业界开源模型清单
- 03 数据（预训练、后训练数据）
- 04 数据合成方法
- 05 模型训练
- 06 模型微调
- 07 模型评估
- 08 模型开源和推广方案
